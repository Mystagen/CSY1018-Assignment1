Glossary

Accessibility

Refers to how easy it is for people with visual impairments and other such handicaps to make use of your website

ASCII

Stands for American Standard Code for Information Interchange and represents all english characters

Attribute

Adds additional information to an element e.g. the alt attribute for an img tag

Bounce

When an email is set back to you due to some issue such as entering an invalid email address

Browser

The software used to translate the HTML pages downloaded from the web

Browser Window

What you are currently viewing in your browser

CSS

Stands for Cascading Style Sheets and used to choose how the website is displayed

Cookies

A small (usually) harmless file that websites store on your PC to simplify certain processes. These include features such as saving user settings

Domain

The name of the place that your website is stored

Domain Name

The web address used to access that domain - e.g. www.google.co.uk

DNS

Stands for Domain Name Server and is the server that redirects the web address you type in to the appropriate ip address

DOCTYPE

Specifies what type of document is being viewed. I.E. DOCTYPE HTML:5 would show the file as have been written in HTML 5

Element

The whole HTML tag from start to end including all attributes

Favicon

Small images displayed in the tab that the website is open in

FTP Client

Software used to transfer files over the web

GIF

A file type used for images. They can store a short video too

Hexadecimal

The base 16 number system. It's one of the ways that is used to represent colour

Hit

A request for a file from a webserver. A hit counter simply shows how many files have been requested

HTML

Stands for HyperText Markup Language and is used to build the actual website. It can't however store any data or be interactive

HTTP

Stands for HyperText Transfer Protocol and handles the transfer of text files accross the internet

HTTPS

Stands for HyperText Transfer Protocol Secure and is a more secure way of accessing webpages

Hyperlink

A link used to travel to another page on the site or another website all together

Hypertext

Any text that includes or consists of hyperlinks

iFrame

Short for Inline Frame and is used to display another web page within the web page

ISP

Stands for Internet Service Provider and its the company that provides your internet connection

IP Address

Stands for internet protocol and is assigned to each computer and router and is used to refer to the machines when sending communications

Javascript

Used to add extra animations and interactivity to a website

LAN

Stands for Local Area Network. A small local network that has its own set of ip addresses all coming from a single router

Landing Page

The page of a website that you first see when you load a website

Link

A connection to another page or another part of the internet

Meta Data

Data contained in the header of a website contained within meta tags

Meta Tag

Tags included in the web pages header thats visible to search engines and is used to locate the site based on search criteria

Operating System

The software that your computer runs that allows you to make use of your machine

Optimise

Either means making the website search engine friendly or compressing videos and images without sarcificing quality

Parked Domain

A domain name that shares a space with another domain

PHP

A linux based language used to increse website functionality and/or link to database

Root

The main folder which all other sub folders or files are stored within

Server

The actual machine that your website is hosted on. You connect to this when you view the website

Server Side

Something that happens or is hosted on the main server as opposed to client side which is hosted on your PC

Script

Code that performs tasks or operations. Commonly refers to javascript

Sitemap

A page that shows all pages on the website and how they all link up. Ususally found in the footer

Subdomain

Allows multiple domains to be hosted on a single domain. These would be accessed with www.subdomain.domain.co.uk

Tag

Is what HTML is built off. Examples include the p tag, the img tag and so on

The Net

Short for the internet. Covers everything in the world thats connected to the world wide web

Trojan Horse

A type of malware that disguises itself as a legitimate program

URL

Stands for uniform resource locator and is the name given to the web address you use to access websites

Website

A collection of one or more HTML pages linked together that you view using a web browser

Website Address

Another name for a URL. What you type into your browser to access a website

WWW

Stands for World Wide Web and is another name for the internet

XML

Stands for Extensable Markup Language and allows the creation of more markup languages
